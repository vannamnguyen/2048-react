# 2048-react

This is the [react](https://reactjs.org/) port of the 2048 game designed by Gabriele Cirulli https://github.com/gabrielecirulli/2048

## Demo

[Demo](https://vannamnguyen.github.io/2048-react/)

## Installation

Install packages:

```
$ yarn install
```

Then run the app:

```
$ yarn start
```
